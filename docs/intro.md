This documentation serves as the comprehensive, code-driven guide for learning advanced concepts in robotics, control theory, and artificial intelligence, with a focus on building complex humanoid systems using ROS 2.

## ðŸ“š Book Structure: Four Pillars of Robotics

The book is organized into four distinct modules, each focusing on a core area of the humanoid robotics stack.

| Module | Core Focus | Key Topics Covered |
| :---: | :--- | :--- |
| **Module 1** | **ROS 2 & Fundamentals** | Setting up the ROS 2 workspace, nodes, topics, services, and the initial URDF for humanoid kinematics. |
| **Module 2** | **Core AI Algorithms** | Path Planning (A*, Dijkstra), Simultaneous Localization and Mapping (SLAM), and foundational Reinforcement Learning (RL) in simulation. |
| **Module 3** | **Control and Planning** | Robotics Sensors, Advanced Computer Vision for environment perception, and Sensor Fusion techniques for robust state estimation. |
| **Module 4** | **Advanced Kinematics & Dynamics** | Homogeneous Transformations, Forward/Inverse Kinematics, PID Control, Robot Dynamics (Lagrangian), and Non-Linear Control (ZMP, Computed Torque). |
